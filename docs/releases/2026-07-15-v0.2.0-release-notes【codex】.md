# One Holy Bible v0.2.0

This public release adds the read-only image-card surface while keeping the code repository small and physically separate from the development workbench.

## Included

- CUV and KJV Scripture for all 66 canonical books, split into book-scoped packages and loaded on demand.
- 10,963 verse-linked explanatory `commentary` and `note` cards.
- 2,705 verse-linked image-card records rendered by the public read-only workbench.
- 2,515 unique content-addressed PNG assets served from the separate [one-holy-bible-assets repository](https://github.com/Siqiho/one-holy-bible-assets).
- Whole-Bible Scripture search, lazy book loading, retry handling, and integrity checks for package and asset metadata.

## Release boundaries

The code repository contains public runtime code, JSON book packages, and image descriptors. Raw image binaries remain in the separate asset repository and are referenced by fixed HTTPS URLs. Private source imports, source PDFs, local paths and services, synchronization internals, review-state metadata, and development-only inputs remain outside this public release.

The MIT license covers project-owned code and documentation only. Scripture, explanatory cards, and image source material retain their own provenance and rights obligations; see [`DATA_SOURCES.md`](../../DATA_SOURCES.md) and [`THIRD_PARTY_NOTICES.md`](../../THIRD_PARTY_NOTICES.md).
