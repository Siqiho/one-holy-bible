# One Holy Bible v0.1.0

This is the first public, text-first release of One Holy Bible.

## Included

- A local-first Bible reading and study workbench.
- CUV and KJV scripture for all 66 books, split into book-scoped packages and loaded on demand.
- 11,124 verse-linked explanatory text cards.
- Whole-Bible scripture search with cross-book navigation.
- Resizable reading and card workspaces, saved layouts, card organization, and local text-card edits.
- Web build configuration, Tauri desktop scaffold, a CLI-driven deterministic public-data generator, automated tests, public-data validation, repository-hygiene checks, and GitHub Actions CI.

## Release boundaries

The image library, private source imports, local service integrations, source PDFs, internal audit material, and personal machine paths are not included in v0.1.0. The published code license does not relicense third-party scripture or explanatory content; see `DATA_SOURCES.md` and `THIRD_PARTY_NOTICES.md` before redistribution.
